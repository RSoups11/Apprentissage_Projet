# Apprentissage_Projet

Ce projet porte sur la classification automatique de tumeurs du sein (bénignes vs malignes) à partir du jeu de données **Breast Cancer Wisconsin (Diagnostic)**.  

L’objectif est de comparer plusieurs méthodes de classification supervisée après une analyse exploratoire et statistique des données.

## Contenu du projet

- `data/` : données brutes, données nettoyées et statistiques descriptives
- `notebooks/` :
  - `exploration.ipynb` : analyse exploratoire des données
  - `modelisation.ipynb` : implémentation et évaluation des modèles
- `rapport/` : rapport final au format PDF
